# Hyperopt Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build an Optuna-based hyperparameter optimizer that tunes Traot's trading parameters by running backtests and minimizing a loss-averse objective function.

**Architecture:** CLI tool at `src/optimize/` that loads historical data once, runs N Optuna trials (each overriding config params and running BacktestEngine), computes loss from trade metrics, and exports best params to config.yaml.

**Tech Stack:** Optuna (TPE sampler), joblib (parallelism), ruamel.yaml (config writing), pandas/pyarrow (data), existing BacktestEngine + BacktestMetrics.

**Spec:** `docs/superpowers/specs/2026-03-14-hyperopt-design.md`

```python
    if metrics.total_trades < 10: return True
    if metrics.win_rate < 0.50: return True
    if metrics.max_drawdown > max_dd_ceiling: return True
    if metrics.largest_loser < -5.0: return True  # pnl_percent is e.g. -5.0
```

```python
def create_objective(
    config_path: str,
    loss_name: str,
    spaces: list,
    symbol: str = None,
    timeframe: str = None,
    data_df: pd.DataFrame = None,
    max_dd_param_active: bool = False,
) -> Callable:
    """
    Create an Optuna objective function.

    Loads data once, returns a closure that:
    1. Suggests params from trial
    2. Overrides config.raw
    3. Runs BacktestEngine
    4. Computes loss
    """
    # Load base config and data ONCE
    raw_config = load_config(config_path)

    if data_df is None:
        # Load from database or download
        data_df = _load_historical_data(raw_config, symbol, timeframe)

    def objective(trial):
        # 1. Suggest params
        params = suggest_params(trial, spaces=spaces)

        # 2. Deep copy config + apply overrides
        trial_config = copy.deepcopy(raw_config)
        apply_params_to_config(trial_config, params)

        # 3. Normalize ensemble weights if applicable
        if 'ensemble' in spaces:
            ensemble_params = {k: v for k, v in params.items() if k.startswith('weight_')}
            normalized = normalize_ensemble_weights(ensemble_params)
            # Apply normalized weights to config
            weights_path = trial_config.get('prediction', {}).get('ensemble', {}).get('weights', {})
            for key, val in normalized.items():
                weights_path[key.replace('weight_', '')] = val

        # 4. Run backtest
        engine = BacktestEngine(config_path=config_path, config_dict=trial_config)
        engine.load_data(df=data_df, copy=False)
        metrics = engine.run_with_model()

        # 5. Compute loss
        max_dd = params.get('max_drawdown_percent') if max_dd_param_active else None
        loss = compute_loss(metrics, loss_name, backtest_days=len(data_df)/(96), max_dd_param=max_dd)

        return loss

    return objective
```
